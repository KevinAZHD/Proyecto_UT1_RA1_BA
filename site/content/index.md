# Pipeline de Datos de Estaciones de Bicicletas

Este sitio documenta un proyecto de ingeniería de datos que procesa información sobre la disponibilidad de estaciones de bicicletas.

## ¿Qué encontrarás aquí?

- **[[docs/1-ingesta-y-almacenamiento|Documentación del Pipeline]]**: Una explicación detallada, paso a paso, de cómo funciona el pipeline, desde la ingesta de datos hasta su modelado.
- **[[reportes/reporte-disponibilidad|Reporte de Disponibilidad]]**: El último reporte generado por el pipeline, con métricas clave sobre la disponibilidad media de bicicletas por barrio.

## Propósito del Proyecto

El objetivo es construir un pipeline de datos robusto y escalable que cubra todo el ciclo de vida de los datos: ingesta, limpieza, almacenamiento en capas (Bronze, Silver, Gold) y generación de un producto de datos final (el reporte).

Puedes navegar por las distintas secciones utilizando el menú de la izquierda.
